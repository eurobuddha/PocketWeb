Features

Everything you get with PocketFS + PocketWeb.

PocketFS — Publish

Decentralized file storage on the Minima blockchain.

📦

On-Chain Storage

Publish ZIP files (up to 50 KB) as blockchain transactions. Permanent, immutable, uncensorable.

🔑

Cryptographic Ownership

Every file is signed with your key. Only you can update your content. Verifiable by anyone.

🔗

Permanent Addresses

Each file gets a permanent Mx address. Share it once — it never changes, even when you update.

🔃

Version Control

Update your published files while keeping the same address. Readers always get the latest version.

🔍

Network Search

Search for files by description. Request files from the network if they're not stored locally.

🔌

Developer API

Other MiniDapps can call PocketFS to load, search, and copy files programmatically.

PocketWeb — Browse

A decentralized browser for the Linternet.

🌐

Browse MiniSites

View websites published via PocketFS. Full HTML, CSS, and JavaScript support.

🔖

Search & Discover

Find sites by keyword or Mx address. Sites spread peer-to-peer through the Minima network.

⭐

Bookmarks

Save your favourite MiniSites. Bookmarks auto-update when new versions are published.

🛠

Site Generator

Build a personal site without code. Choose colours, add a blog, pick fonts — publish in one click.

🧰

Linked Sites

MiniSites can link to each other and share resources. A real interconnected decentralized web.

💻

Developer Mode

Full read-only MDS API access. Build data-driven apps that query the blockchain directly.

Ready to try it?

Install both apps on your Minima node in under 5 minutes.
