Run a Node

Get Minima running on any platform

Android

  1. Install from Google Play Store (or download APK from GitHub)
  2. Allow battery optimisation exemption for background operation
  3. Enable auto-update in Play Store settings
  4. Write down your seed phrase immediately
Huawei/Redmi/ZTE: Disable "Manage Automatically" in battery settings. OPPO: Keep background apps under 5.

Desktop Connect: Access your phone node from a computer on the same WiFi via Settings → Desktop Connect.

Java JAR (Easiest Desktop Method)

The simplest way to run Minima on any desktop. Just need Java installed and the minima.jar file.

Mac / Linux (Terminal)

java -jar minima.jar -data ~/.minima -mdsenable -mdspassword YOUR_PASSWORD

Windows (Command Prompt)

java -jar minima.jar -data %USERPROFILE%\.minima -mdsenable -mdspassword YOUR_PASSWORD

Then connect to the network:

peers action:addpeers peerslist:megammr.minima.global:9001

Access the MiniDapp Hub at https://127.0.0.1:9003

Common Startup Parameters

ParameterDescription
-data [path]Where to store node data
-mdsenableEnable the MiniDapp System
-mdspassword [pass]Set the MDS login password
-rpcenableEnable RPC interface on port 9002
-cleanWipe data and start fresh
-port [num]Set base port (default 9001)
-connect [ip:port]Connect directly to a specific peer
-archiveRun as an archive node (keep full history)

Running in Background

Mac/Linux:

nohup java -jar minima.jar -data ~/.minima -mdsenable -mdspassword YOUR_PASSWORD &

Windows: Use javaw instead of java to run without a console window:

javaw -jar minima.jar -data %USERPROFILE%\.minima -mdsenable -mdspassword YOUR_PASSWORD

Mac (Open Web Start)

  1. Install Open Web Start (Intel: macos-x64, M1/M2: macos-aarch64)
  2. Download JNLP file from spartacusrex.com
  3. Create desktop shortcut (Make Alias) and set to launch on startup
  4. Set password via Settings → Advanced params → -mdspassword YOUR_PASSWORD
Always shut down Minima before shutting down your Mac to prevent database corruption.

Docker

docker run -d --name minima \
  -e minima_mdspassword=YOUR_PASSWORD \
  -e minima_desktop=true \
  -v minima-data:/home/minima/data \
  -p 9001-9003:9001-9003 \
  --restart unless-stopped \
  minimaglobal/minima:latest

Auto-update with Watchtower:

docker run -d --name watchtower \
  -v /var/run/docker.sock:/var/run/docker.sock \
  containrrr/watchtower \
  --interval 28800 minima

Login at https://localhost:9003 with your password.

First Steps After Install

  1. Join the network — your node syncs automatically
  2. Lock your wallet — set a password in the Security MiniDapp
  3. Write down your seed phrase — this is your backup (see Guides)
  4. Set your profile — optional, for Maxima contacts
