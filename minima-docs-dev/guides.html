User Guides

Essential guides for securing and managing your node

Secure Your Seed Phrase

Your seed phrase is the master backup for your entire wallet. If you lose your device, the seed phrase is the only way to recover your coins.

Finding Your Seed Phrase

  • GUI: Security MiniDapp → Manage Seed Phrase → Show Seed Phrase
  • CLI: Run the vault command in the terminal

Best Practices

  • Write it on paper or a metal plate — never screenshot or copy/paste on a connected device
  • Make 2 copies stored in separate physical locations
  • Give one copy to a trusted relative for disaster recovery
  • Never store it digitally — no photos, no cloud, no notes apps

Lock Your Wallet

Set a password in the Security MiniDapp to encrypt your private keys. Without a password, anyone with device access can spend your coins.

Backup Your Node

Regular backups protect against device failure. Use the Security MiniDapp to create encrypted backups that can be restored on a new device.

A backup is different from a seed phrase. Backups include your MiniDapp data, bookmarks, and settings. The seed phrase only recovers your coins.

Node Recovery Options

MethodWhat It Recovers
Restore BackupEverything — coins, MiniDapps, settings, contacts
Import Seed PhraseCoins only — fresh node with your wallet
Chain Re-sync (QuickSync)Fixes sync issues — re-downloads chain from peers

Create a Safe

For large amounts, create a Safe — a separate seed phrase and key set on the same node. This isolates high-value coins from daily-use keys.

Cold Storage

For maximum security, sign transactions offline:

  1. Run one node offline (air-gapped) — this holds your keys
  2. Run one node online — this broadcasts transactions
  3. Construct the transaction on the online node
  4. Transfer to the offline node (USB stick) to sign
  5. Transfer signed transaction back to the online node to broadcast
