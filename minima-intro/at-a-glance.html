Below you will find key information about Minima, with links to further reading.

Layer 1 Protocol

{/ Implementation Language: Java
/}

{/* Consensus mechanism: TxPoW - Transaction Proof-of-Work

Longest chain: GHOST Protocol (heaviest chain)

Hashing algorithm: SHA3, considered to be Quantum resistant

Transaction Model: UTxO - Unspent Transaction Output (same as Bitcoin)
*/}

{/* ### Transactions/Blocks

Txns/block: 256

Txns/second: ~5 on layer 1, unlimited on layer 2

Block intervals: 50 seconds
*/}

The Minima Coin

{/ Native currency: Minima coin
/}

{/* Supply: 1 billion hard-capped, deflationary via the Burn

Custom tokens: supported natively via coloured coins, minted directly on node

NFTs: supported natively via coloured coins, minted directly on node
*/}

Smart Contracts

{/ Minima has a powerful scripting language, KISS VM, for writing Smart Contracts. /}

Layer 2 Scaling

{/ Omnia unlocks virtually unlimited transactions per second via off-chain payment channels, similar to Bitcoin's Lightning network. /}

Off-Chain Communication

{/ Maxima provides off-chain, encrypted, peer-to-peer communication. /}

Web3 DApps

{/ MiniDapps are Minima's decentralized applications, built using widely known languages including HTML, CSS, JavaScript, React etc /}
