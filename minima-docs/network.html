Network Overview

Four layers powering a truly decentralized network

Layer 1 — Minima

On-chain value transfer. All transactions are processed by all nodes (flood-fill). Every node collectively comes to consensus on the blockchain state. This is the trust layer — where users initiate relationships and settle disputes.

Maxima — Information Layer

Off-chain information transfer. Point-to-point encrypted messaging running over the Minima peer network. Every Minima node is also a Maxima node.

  • End-to-end encrypted using each user's Maxima public key
  • No additional infrastructure — your node is your messaging endpoint
  • Free to use, with ~1 second TxPoW to prevent spam
  • Enables censorship-resistant messaging apps and P2P data exchange
  • Used by MiniFS to fetch files from peers before falling back to on-chain requests

Layer 2 — Omnia

Off-chain value transfer. Fast, cheap, and scalable payments via bi-directional payment channels using ELTOO technology (more advanced than Bitcoin's Lightning).

  • Users set up channels on Layer 1, then transact off-chain indefinitely
  • Supports smart contract sequences with subsets of users and coins
  • Hash Time-Locked Contracts (HTLCs) enable payment routing
  • Settle on Layer 1 whenever needed
  • Essentially unlimited transactions per second

Layer 3 — MiniDapps

Web3 applications. Decentralized apps built with HTML, CSS, and JavaScript that combine all lower layers:

  • Value transfer via Minima (Layer 1) and Omnia (Layer 2)
  • Information transfer via Maxima
  • Smart contracts via KISS VM (Turing Complete)
  • Front-end uses standard web technologies — accessible to any web developer
  • Packaged as .mds.zip files and installed directly on your node
You are reading this page inside a MiniDapp right now — PocketWeb is a Layer 3 application browsing content stored on Layer 1.
