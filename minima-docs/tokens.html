Tokens & Tokenomics

Minima's coin supply, custom tokens, and NFTs

Tokenomics

Minima has a hard-capped supply of 1 billion coins that will slowly reduce over time as a result of The Burn. All coins were minted at the Token Generation Event (TGE) in Q1 2023.

100% of native Minima (1,000,000,000 $MINIMA coins) were minted at the TGE. 12.5% of these coins are wrapped on the Ethereum blockchain as ERC-20 tokens (125,000,000 $WMINIMA coins), enabling trading on exchanges ahead of native CEX integration. Wrapped Minima can always be exchanged 1:1 with native Minima via the Minima Treasury.

Circulating Supply

Post-TGE tokenomics significantly reduced circulating supply — 55% reduction at 18 months vs. the original plan. Full supply unlocks over 6 years post-TGE. At 2 years: 43% circulating. At 3 years: 62%. At 6 years: 100%.

Maximize

Users can earn yield on native Minima via the Maximize MiniDapp — a fully collateralised, non-custodial lockup. Yield is paid in native Minima to participants who lock their holdings for set periods, encouraging long-term node running and reducing circulating supply.

Grants & Bounties

Grants are available for building novel use cases on Minima, awarded to developers and entrepreneurs. Bug bounties are available for critical bugs reported via Discord.

Coin Utility

The Minima coin is required to use the network in a purposeful way. Main drivers:

  • Store of Value: Hard-capped, deflationary via The Burn, PoW-secured. Cannot be inflated like fiat.
  • Medium of Exchange: All users running nodes can transact directly, peer-to-peer, anywhere, anytime.
  • The Burn: Transaction fees are permanently destroyed, making the currency more scarce over time.
  • Tokenisation: Minima is required to create custom tokens and NFTs, which inherit the same security as the base chain.
  • MiniDapp Usage: Payments, DeFi, DEXs, DAOs, gaming, voting, social media, funding — unlimited applications.
  • Partnerships: Enterprise and DePIN integrations increase credibility and demand for the network.

Coloured Coins

Minima is the native coin for the Minima blockchain. Each UTxO is defined as a coin and therefore can be worth any amount. Minima supports custom tokens (including NFTs) natively. Tokens are "Coloured Coins" — tiny fractions of Minima which represent the supply of a custom token or NFT.

Token Attributes

AttributeDescription
TokenIDCreated by hashing the CoinID and total amount. Globally unique.
NameA string or full JSON with name, description, icon URL, etc.
CoinIDThe CoinID used when creating the token initially.
TotalThe total number of these tokens. Immutable after creation.
DecimalsNumber of decimal places. 0 for NFTs (indivisible).
ScriptOptional token script. Default is RETURN TRUE.
Total AmountThe total Minima coloured to represent this token.
ScaleThe scale factor between Minima amount and token supply.

Since Minima uses 44 decimal places, if you colour 1×10-33 Minima, that creates 1,000 tokens with 8 decimal places (scale = 36).

NFTs

NFTs are simply custom tokens with 0 decimal places — they can only be spent whole, exactly like a physical collectible. They inherit the full security of the Minima blockchain.

Creating Tokens

tokencreate name:MYTOKEN amount:1000 decimals:4

With JSON metadata:

tokencreate name:{"name":"MyNFT","description":"A unique item","icon":"https://..."} amount:1 decimals:0
Custom tokens can be transferred with the same send command as Minima — just add tokenid:0xFF... to specify the token.
