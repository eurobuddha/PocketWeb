Maximize

A non-custodial MiniDapp that lets users earn yield on native Minima by locking coins for set periods — reducing circulating supply while rewarding long-term holders.

Coloured Coins

Custom tokens and NFTs are supported natively on Minima — no smart contract deployment needed.

How It Works

A tiny fraction of Minima is "coloured" to represent a new token. The coloured amount determines the total supply based on a scale factor:

  • Minima uses 44 decimal places internally
  • Colouring 1×10-33 Minima creates 1,000 tokens with 8 decimal places
  • Each token gets a globally unique TokenID (hash)

Token Properties

PropertyDescription
TokenIDUnique hash identifier
NameString or JSON metadata (name, description, icon URL)
Total SupplySet at creation, immutable
Decimals0 for NFTs (indivisible), up to 44
ScriptOptional spending rules (default: RETURN TRUE)

NFTs

An NFT is simply a custom token with 0 decimal places — it's indivisible and must be spent whole, exactly like a physical collectible.

Creating Tokens

tokencreate name:MYTOKEN amount:1000 decimals:4

With JSON metadata:

tokencreate name:{"name":"MyNFT","description":"A unique item"} amount:1 decimals:0
