Core Features
What makes Minima unique

UTxO & TxPoW

Bitcoin-style transaction model where every user mines their own transactions

Cascading Chain

Logarithmic pruning keeps the chain small enough for mobile devices

GHOST Consensus

Heaviest-chain selection with adaptive difficulty and 50-second blocks

Quantum Secure

Winternitz signatures with SHA3-256 — resistant to quantum attacks

4-Layer Network

Minima + Maxima + Omnia + MiniDapps — value, data, speed, apps

Native Tokens & NFTs

1 billion hard-capped supply, custom tokens via coloured coins
Who is Minima for?
Creating value for four groups

Enterprises

Deploy lightweight nodes that scale with users. Reduce infrastructure costs. No specialised hardware.

DePIN Projects

Access a large decentralised P2P network. Deploy on mobile and IoT. Quick proof-of-concept.

Developers

Run a full node locally. Build MiniDapps with HTML/CSS/JS. Instant private testnet.

Individuals

Trustless P2P payments. Tokenise real-world assets. Participate in block construction.
