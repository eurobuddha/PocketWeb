What is MiniFS?

MiniFS lets you publish small ZIP files across the Minima network as regular transactions. Each file gets a permanent Mx.. address — like a web address, but on-chain. Anyone running MiniFS can find and open your files, and MiniWEB can browse them as websites.

50k maxZIP format only
Permanent addressOnly you can update it
DecentralisedNo servers, lives on-chain

Getting Started

1 — Create a File

  1. From the home screen tap New MiniFS File
  2. A unique Mx.. address is automatically generated for you
  3. Write a description — this is how others will discover your file
  4. Tap Load ZIP File and choose your ZIP (max 50k)
  5. Tap Create MiniFS File — the file is saved locally and marked unpublished

2 — Publish to the Network

  1. From the home screen tap Publish All
  2. Confirm — each file needs 1 on-chain transaction
  3. Once the transaction confirms, your file is live on the Minima network
  4. To release a new version, edit the file and publish again — the Mx.. address stays the same

3 — Search & Browse

  1. Tap Search in the top bar to find files by description or Mx.. address
  2. Tap Browse on any result to open it in MiniWEB
  3. If a file isn't found locally, MiniFS automatically requests it from your Maxima contacts — or broadcasts an on-chain request if needed
Tip: Having Maxima Contacts improves file discovery. Files can be sent peer-to-peer over Maxima before falling back to an on-chain request.

For Developers

Other MiniDAPPs can load and search files in MiniFS using the MDS API. Call MDS.api.call("minifs", ...) with one of the supported actions below.

Supported Operations

LOAD Load a file packet by its Mx.. address. If not found locally, a network request is triggered automatically.
SEARCH Search all locally known files by description text. Returns a list of matching file packets.

ZIP Extraction

Once you have the file packet data, use these two libraries to extract the ZIP contents into the MDS file system:

ziputil.js jszip.min.js

extractZIP(zipdata, mdsfolder, callback) — include both files and call this function with the hex ZIP data from the file packet.

MiniWEB uses exactly this system to browse MiniFS files as websites.
